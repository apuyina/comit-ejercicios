Documentación

Autentificarse

  • Endpoint: /login
  • Método: GET
  • Respuesta: token para autentificarse

Mensajes

Leer

  • Endpoint: /mensajes
  • Método: GET
  • Headers
    • token: [token obtenido al autentificarse]
  • Respuesta: lista de mensajes

Crear

  • Endpoint: /mensajes
  • Método: POST
  • Headers
    • token: [token obtenido al autentificarse]
  • Respuesta: Identificador del mensaje

Modificar

  • Endpoint: /mensajes
  • Método: PATCH
  • Body: texto
  • Headers
    • token: [token obtenido al autentificarse]
    • id: [identificador del mensaje]
